<h1> Idea framework 超轻量级框架</h1>
<h3>Idea核心理念</h3>
* 用户体验目标，简单到极致！
* 降低框架学习成本，以最好的通用性开发低耦合的产品。

<h3>框架功能简述</h3>
* 超轻量级框架，拥有框架应有的核心功能。
* 完全基于OOP思想、MVC架构开发
* 采用PDO，摒弃ORM，兼容PHP7
* 支持自定义项目结构，入口绑定支持（多入口）。
* 完善的路由支持：普通模式、PATHINFO、伪静态等。
* 内置模板引擎，具有缓存、编译、语法机制。
* 数据层基于PDO，模型对象单例实现，CURD更贴近原生，无须深入学习框架
